Variables con Css

Crear clase que contenga las variable

para usar las varibles se debe crear una clase que contenga las variable con los colores en este caso le llamomos color

Usar la variable

Usamos var(--nombreDeLaVariable);

Uso en el HTML

en el html llamamos a la clase donde usamos la variable y la clese que contiene todas las variables, en el caso de los titulos usamos las clases titulo y color
