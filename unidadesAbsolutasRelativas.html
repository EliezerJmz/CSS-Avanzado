Unidades Relativas em

Toman como base las unidades del contenedor padre

el padre tiene como base 20px

el hijo tiene 1.5 em lo que multiplicara por 1.5 * 20px del padre = 30px

Texto Padre 20px
Texto hijo 1.5em
Hereda el tamaño del texto padre 20px
Texto de 2em = 40px
Texto Sub hijo tomo como base a su contenedor padre
Texto sub hijo 2em por la base del padre 40px = 80px

Unidades rem

Toman siempre como base principal 16px que usan los navegadores por defecto

Texto padre base 20px
Texto 2rem = 16px tamaño navegador por default * 2 = 32px
Texto sub hijo 2rem Mantiene la base del navegador de 16px
