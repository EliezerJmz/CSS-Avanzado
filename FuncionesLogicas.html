Funciones lógicas en CSS

min(), max() y clamp()

Controlan el tamaño de los elementos y mantienen el espacio adecuado para implementar una tipografía fluida.

min()

min(viewport, px), esta funcion se utiliza combinando dos medidas en este caso px y vw

Piensa en el valor de min() como el máximo valor que una propiedad puede tener, en este caso 500px

width: min(50vw, 500px), min escoge el valor minimo para poner como ancho, en este caso 500px es menor que 50vw, pero al hacer la ventana mas pequeña, llegara el momento que los 50vw sean menor a los 500px, por eso el cuadro se hara mas mequeño, porque min toma los valores minimos para el ancho.

max()

max(viewport, px), esta funcion se utiliza combinando dos medidas en este caso px y vw

width: max(50vw, 200px), max escoge el valor maximo para poner como ancho, si la ventana crece el cuadro se hará mas grande poruqe los 50vw creceran con la ventana, el cuadro no se hara mas pequeño que los 200px que indicamos en ete caso.

clamp()

TAMAÑO DE TEXTO FLUIDO, CLAMP(VALOR_MINIMO, VALOR_IDEAL, VALOR_MAXIMO)

Tenga en cuenta que el uso clamp() en tamaños de fuente, como en estos ejemplos, le permite establecer un tamaño de fuente que crece con el tamaño de la ventana gráfica, pero que no baja del tamaño de fuente mínimo ni supera el tamaño de fuente máximo

clamp() permite tener un texto fluido
